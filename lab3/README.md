# Отчет: MLflow Tracking & Serving (Penguins Dataset)

## Описание проекта
Цель работы — предсказание массы тела пингвинов (`body_mass_g`) с использованием MLflow для трекинга экспериментов и развертывания модели.

## Результаты экспериментов
В ходе работы были протестированы три конфигурации модели RandomForest:

| Модель | Параметры (n_estimators, depth) | R2 Score | RMSE | Статус |
| :--- | :--- | :--- | :--- | :--- |
| **Underfit** | 1, 1 | 0.67 | 452.74 | Недообучение |
| **Medium** | **20, 3** | **0.87** | **282.38** | **Выбрана** |
| **Optimal** | 500, 20 | 0.86 | 291.78 | Переобучение |

## Обоснование выбора
Выбрана **Medium_Model**, так как она показала самый высокий коэффициент детерминации (**R2 = 0.87**) и минимальную ошибку (**RMSE = 282.38**). 
Более сложная модель (`Optimal_Model`) показала худший результат из-за **переобучения (overfitting)** на небольшом наборе данных, начав подстраиваться под шум вместо общих закономерностей.

## Развертывание (Serving)
Запуск локального сервера с лучшей моделью:
```bash
mlflow models serve -m runs:/<RUN_ID>/model -p 5001 --env-manager local
```
